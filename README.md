# 🇮🇷 Mirava — Free Mirrors & Connectivity Dashboard

**Mirava** gives you fast, free, and uncensored access to Linux mirrors, programming libraries, and essential tools. Track mirror status in real-time and contribute to keeping the network reliable.

---

## 🌐 Features

- Browse **official Iranian mirrors** and **global mirrors**.  
- Real-time **connectivity dashboard** with online/offline status.  
- Copy mirror URLs to clipboard with one click.  
- Track **mirror history** and see trends over time.  
- Community-driven: contribute updates and improvements.  

---

## 📦 How to Use

### 1. Browse Mirrors
- Mirrors are displayed in a **grid view** with description and available packages.  
- Click any mirror card to **copy the URL**.  

### 2. Check Mirror Status
- The dashboard shows **real-time connectivity**.  
- Status colors:  
  - 🟢 Online  
  - 🔴 Offline  

### 3. Contributors
- Mirava is built by a global community.  
- View all contributors in the **Contributors section**.  

---

## 🛠️ Contributing

If you want to report or add a mirror:

1. **Fork the repository**  
2. Edit the `mirror.json` file  
3. Submit a **Pull Request**  

> ⚠️ Status may vary depending on your ISP. All reports are community-verified.  

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Data:** `mirror.json`  
- **Live Status:** Fetch API with periodic checks (every 15s)
